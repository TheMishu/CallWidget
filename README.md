# Instalacja:
1. Ściągnąć zip z repo na githubie i go rozpakować.
2. Konsola -> cd Backend
2.1 >npm install
2.2 >node app
3. Konsola -> cd Frontend 
3.1 >npm install
3.2 >ng serve
4. Wejść na localhost:4200 i nacisnąć słuchawkę
